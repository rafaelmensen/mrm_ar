# MRM AC App

Projeto Vite + React + TypeScript + Tailwind pronto.

## Rodar

```powershell
cd mrm-ac-app
npm install
npm run dev
```
Abra o endereço que aparecer (por padrão http://localhost:5173).

Se você já tinha um `postcss.config.js`, **apague** e usei este `postcss.config.cjs` para não dar erro de ESM.
